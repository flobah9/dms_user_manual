Introduction

The Delivery Management System (DMS) is a comprehensive platform designed to streamline the process of placing, tracking, and managing deliveries. Whether you're a customer, delivery driver, or administrator, this user manual will guide you through the functionalities of the system to ensure a seamless experience.

Getting started

Installation

To install, navigate into the folder and run npm install.
After installation connection delivery_management.sql database to server. Then run the following command npm start.
To access the DMS, visit http://localhost:3000 from your device. The login screen will appear.

Order Management

Order management allows admin to create, track, and manage orders efficiently.

Features:

  • Create new orders
  • Edit existing orders
  • Cancel orders
  • View order history

Fleet Management

Fleet management helps in managing delivery vehicles and drivers.

Features:

  • Add new vehicles
  • Assign drivers to vehicles
  • Track vehicle maintenance
  • Monitor fuel consumption

Employee Management

Employee management module is used to manage delivery staff and other personnel.

Features:

  • Add new employees
  • Assign roles and permissions
  • Monitor employee performance
  • Manage payroll

Tracking Order

Tracking order feature enables users to track the status and location of orders in real-time.

Features:

  • Real-time order tracking
  • Delivery status updates
  • Estimated time of arrival (ETA)
  • Notifications for delays or changes

Customer Management

Customer management allows users to manage customer information and interactions.

Features:

  • Add new customers
  • View customer details
  • Manage customer accounts
  • Track customer orders and feedback

Report Issues

In the event of any issues or concerns during the delivery process, customers can report them directly through the DMS. Our support team will promptly address any issues and provide assistance as needed.
